## UX Rules
- Minimum 44x44px touch targets
- Visible focus rings on all interactive elements
- Use loading states for async actions
- Avoid layout shift on hover

## Anti-patterns
- Low-contrast text
- Emoji icons in UI
- Inconsistent card padding
